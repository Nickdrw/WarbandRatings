# Warband Ratings

Inspired by a WeakAura I used back in the WeakAura era, **Warband Ratings** is a very lightweight addon that lets you check your ratings no matter which character you are currently logged into.

It provides an instant overview of your **PvP** and **PvE** ratings, organized by **class** and **character name**, so you can quickly see where each of your characters stands across your warband.

## Supported ratings

- Solo Shuffle
- Solo Battlegrounds
- 2v2
- 3v3
- 10v10
- Mythic+

## How it works

When a character is logged in, the addon records and updates that character’s rating data automatically.  
Over time, this builds a shared database that you can consult from any character.

## Goal

The purpose of **Warband Ratings** is simple: make it easy to track all your characters’ ratings without having to log in and out repeatedly or rely on external tools.
